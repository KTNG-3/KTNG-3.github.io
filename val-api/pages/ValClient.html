Valorant Client API

Authentication

First We Need To Login To Valorant Account

const ValAuth_Account = await ValApi.ValClient.Auth.login('USERNAME', 'PASSWORD'); But We Have 2 Type Of Account

Multi-Factor Authentication

When The Account Have Multifactor You Can't Use Normal Method

After Login The Verify Code Will Send To Mail

const VerifyCode = 123456; // <---------- Verify Code
const ValAuth_Multifactor = await ValApi.ValClient.Auth.verify(ValAuth_Account, VerifyCode);

Client

After We Get Account, Then We Will Create an API Client

const ValAPI_Client = new ValApi.ValClient(
    ValAuth_Multifactor,  // Use ValAuth_Account When It Isn't use Multifactor
);
